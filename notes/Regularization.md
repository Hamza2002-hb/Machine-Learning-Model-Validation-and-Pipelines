# Regularization

Regularization helps reduce overfitting.

It adds a penalty to large coefficients.

This forces the model to become simpler.

---

## Ridge Regression

Uses L2 Regularization.

Shrinks coefficients but never makes them zero.

Useful when all features are important.

---

## Lasso Regression

Uses L1 Regularization.

Can shrink coefficients exactly to zero.

Useful for feature selection.
