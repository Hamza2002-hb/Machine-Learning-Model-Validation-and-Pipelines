# Model Validation

## What is Model Validation?

Model Validation is the process of checking how well a machine learning model performs on unseen data.

The goal is to make sure the model does not memorize the training data and can make accurate predictions on new data.

---

## Why is Model Validation Important?

Without validation, the model may perform well only on training data but fail on new data.

Validation helps us measure the true performance of a machine learning model.

---

## Types of Data

Training Set

Used to train the model.

Validation Set

Used to tune hyperparameters.

Test Set

Used only once to evaluate the final model.

---

## Generalization

Generalization means how well a model performs on unseen data.

A model with good generalization performs well on both training and testing data.

---

## Overfitting

Overfitting happens when the model memorizes the training data instead of learning patterns.

Such a model performs poorly on unseen data.
